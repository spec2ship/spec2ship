**Status**: draft
**Priority**: low
**Category**: Output
**Created**: 2026-01-11

## Context

After a roundtable produces artifacts, users may want to export them in different formats without re-running the roundtable.

## Proposal

### 1. Export Commands

- `/s2s:specs:export --format ieee830`
- `/s2s:design:export --format arc42`

### 2. Formats Supported

**Specs**:
- IEEE 830 (Software Requirements Specification)
- Volere (requirements template)
- Simple list

**Design**:
- Arc42 (architecture documentation)
- C4 (context, containers, components, code)
- Simple list

### 3. Source

- Read from session file or requirements.md
- Transform to target format
